nwpc-operation-system-tool
=====

NWPC 业务系统工具套件，包括以下组件：

* SMS 作业提交脚本
* 业务系统分析工具

简介
----

### SMS 作业提交脚本

将 SMS 作业脚本提交到 LoadLeveler 的脚本，检测提交是否成功。

### 业务系统分析工具

分析SMS作业提交失败日志的桌面应用软件。


具体说明请参阅各个组件的 README.md 文件。
